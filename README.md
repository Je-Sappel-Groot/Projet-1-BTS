# Projet 1 – BTS SIO SLAM (E6)
## Training Academy – Gestion d’une académie de formation

### Contexte
Ce projet a été réalisé dans le cadre du BTS SIO option SLAM, épreuve E6.
Il s’agit d’une application web permettant la gestion d’une académie de formation
avec utilisateurs, enseignants, étudiants, cours, notes et contacts.

### Objectifs
- Authentification sécurisée des utilisateurs
- Gestion des enseignants et étudiants
- Gestion des cours
- Attribution et consultation des notes
- Tableau de bord récapitulatif
- Respect d’une architecture MVC
- Utilisation d’une base de données relationnelle

### Technologies utilisées
- Node.js
- Express.js
- MySQL
- Sequelize (ORM)
- JWT (authentification)
- Bcrypt (hash des mots de passe)
- EJS (vues)
- Bootstrap (UI)

### Base de données
Scripts SQL disponibles dans le dossier `sql/`
